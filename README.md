# mosamatic3
Command-line version of Mosamatic tool for body composition analysis

## commands
The tool "mosamatic3" supports the following commands:
* help 
* pipeline
* sliceselect
* rescale
* segment
* scores
* dcm2png
* seg2png
* seg2nii
* nii2seg

## help
Usage: mosamatic3 help

## pipeline
Usage: mosamatic2 pipeline

## sliceselect
Usage: mosamatic2 sliceselect

## rescale
Usage: mosamatic2 rescale

## segment
Usage: mosamatic2 segment

## scores
Usage: mosamatic2 scores

## dcm2png
Usage: mosamatic2 dcm2png

## seg2png
Usage: mosamatic2 seg2png

## seg2nii
Usage: mosamatic2 seg2nii

## nii2seg
Usage: mosamatic2 nii2seg
